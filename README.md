# Customer Queue System

This is a C program that implements a Queue data structure using a Linked List to manage a customer waiting line.

## Features
- Dynamic memory allocation using `malloc`
- Add customers to the queue (`enqueue` operation)
- Display the waiting line
- Implements FIFO (First-In-First-Out) logic

## Technology
- Language: C
- Concept: Queue, Linked List, Pointers & Dynamic Memory
